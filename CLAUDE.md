# プロジェクト概要
WordPressカスタムテーマの開発

## テーマ情報
- テーマ名: my-theme（適宜変更）
- 対象: コーポレートサイト / ブログ / etc.
- カラー: メインカラー #1e3a8a / アクセント #dc2626

## コーディング規則
- クラス名プレフィックス: `mt-`（テーマ略称）
- CSSはカスタムプロパティ（変数）を使う
- PHPはWordPressコーディング規約に従う
- ACFを使用する

## Step 2：Codespace を開いてターミナルで Claude に依頼
```
まず WordPressテーマの基本ファイル構成を作ってください。
テーマ名は「my-theme」で、シンプルなコーポレートサイト用です。
必要なファイルをすべて生成してください。
```

Claude Code が以下を自動生成してくれます：
```
theme/
├── style.css          ← テーマ情報
├── functions.php      ← 機能定義
├── index.php
├── header.php
├── footer.php
├── front-page.php
├── page.php
├── single.php
├── archive.php
└── assets/
    ├── css/
    └── js/
```
